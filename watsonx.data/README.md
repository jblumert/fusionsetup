## install the cpd-cli:

- edit cpd_var.sh
- source cpd_var.sh
- get cpd_cli:
  - https://github.com/IBM/cpd-cli/releases
  - install cpd_cli and setup $PATH
    - bashrc
  - source .cpd_var.sh in .bashrc 
  - run ./install.sh

#### linux version for x86:
 - https://github.com/IBM/cpd-cli/releases/download/v13.1.5/cpd-cli-linux-EE-13.1.5.tgz
